# More Than Todo

A local-first task, planning, project, and focus timer web app built with Vite,
React, TypeScript, Tailwind CSS, Zustand, and IndexedDB.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is emitted to `dist/`.
